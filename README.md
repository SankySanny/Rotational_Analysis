# Rotational_Analysis
This is a Detailed Volleyball Rotational Analysis app. Pulls data from a .dvw file format
Run it in R and just upload a .dvw file to see detailed rotational information
Can filter by sets
Calculations explained: Peak rotations are calculated based on attach eff, it can be modified to calculate peak rotations based on SO%. 
Calculations explained: MVP badge is based on the top point scorer for each team.
